# Skeleton-Dactyl-Mini

Compact and silent Dactyl-like keyboard for ergo lovers.


# Versions

V1:

- original version

V2:

- smoother
- should work with amoebas and flexible pcbs
- more screws !

V2.7:

- compatible with the [elite-c holder pcb](https://github.com/Bastardkb/Elite-C-holder)


# Parts list
